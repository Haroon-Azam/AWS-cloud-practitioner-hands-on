## Objective
To create a reusable Amazon Machine Image (AMI) from an EC2 instance and understand instance templating in AWS.

---

## AWS Services Used
- Amazon EC2
- Amazon EBS
- Amazon AMI

---

## Tasks Performed
- Created an AMI from an existing EC2 instance
- Generated EBS snapshots automatically during AMI creation
- Verified AMI availability
- Launched a new EC2 instance from the custom AMI 

---

## Why AMIs are Important
AMIs enable rapid deployment, scaling, and consistency by allowing users to launch multiple identical EC2 instances.

---

## Outcome
- Successfully created and used a custom AMI
- Understood reusability and backup concepts in AWS
